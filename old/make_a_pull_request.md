# プルリクを企む

```bash
$ git clone https://github.com/name.git
$ cd .\name\
$ git checkout -b 1st
$ git branch -a
$ git pull origin 1st:1st
$ git checkout -b ""
$ git branch -a
```

編集<br>

```bash
$ git status
$ git add .
$ git commit -m "typo 2 letters '"
$ git push -u origin ""
```

自分が所有するリポジトリに対して上記コマンドが有効であることを確認
